Hanki Anki Helper - Draft Notes for Anki

Think first. Sync later.

Notes first. Cards later. Hanki Helper is a browser extension that allows you to take draft notes right in the browser and later sync to Anki or delete, all without leaving the browser.

Frequently Asked Questions

  • Do I need to create an account?

    No, as soon as you install Hanki Anki Helper, you will be able to start saving notes. Syncing notes to anki requires the Anki app to be installed

    Learn more

  • What data is collected?

    No, as soon as you install Hanki Anki Helper, you will be able to start saving notes.

  • What is AnkiConnect?

    AnkiConnect enables external applications (like Hanki) to communicate with Anki over a simple HTTP API.

    Learn more

  • Where are my notes stored?

    Notes are stored locally on your machine using Chrome's local storage

    Learn more

  • I deleted a note on Hanki, can I get it back?

    No. Currently notes that are deleted are gone forever.

    Learn more
